To run:
First of all, download the repository.
 Then:
- For the react website - Check that you have npm installed using npm -v, bootstrap and bootstrap-icons installed.
  For bootstrap: npm install bootstrap and npm install bootstrap-icons. Then, using npm install and then npm start in cmd (when you are in the directory of the web project)
  you can run it in your default browser.
- For the Android app, open the myApplication directory in Android Studio and run using an emulator.
