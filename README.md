This is a project that contains a 'YouTube' like website and Android app. We developed the website using React with Bootstrap and the app using Android Studio.
One of us did some pages in React and Android and the other did the remaining pages, and we helped each other during the progress. We worked using a few git branches that we merged according to our progress and procedure.

## How to Run

### React Website
1. Install Node.js and npm.
2. Open a terminal in the `wetubeweb` folder.
3. Run `npm install`.
4. Run `npm start`.
5. Open the app in your browser at `http://localhost:3000`.

### Android App
1. Install Android Studio.
2. Open the `MyApplication` folder in Android Studio.
3. Let Gradle sync finish.
4. Run the app on an emulator or a connected Android device.
